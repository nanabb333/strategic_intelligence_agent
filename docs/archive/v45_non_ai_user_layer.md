# V4.5 Non-AI User Layer

V4.5 adds a bilingual product layer for users who want structured strategic intelligence without learning prompt engineering.

## What Changed

- Added English, Simplified Chinese, and Traditional Chinese locale files.
- Added guided questions mapped to internal analysis goals.
- Added deterministic output adaptation for beginner, analyst, and executive modes.
- Updated the local dashboard with language selection, guided question buttons, and output mode selection.
- Added non-AI user guides and localized beginner examples.

## What Did Not Change

The core pipeline remains a decision-support workflow:

Document -> Issue Extraction -> Scenario Classification -> Historical Retrieval -> Context Retrieval -> Mechanism Detection -> Multi-Lens Analysis -> Evidence Assessment -> Executive Brief

## Guardrails

The system does not provide forecasts, probabilities, investment advice, trading recommendations, or certainty claims.

## Portfolio Value

This layer demonstrates AI product design beyond model prompting: user segmentation, localization, workflow simplification, evidence traceability, and portfolio-ready demo packaging.
