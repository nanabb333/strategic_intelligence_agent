# 非 AI 用户指南

这个工作台面向不会写提示词的用户。选择语言，粘贴或上传文档，选择一个引导问题，再选择输出模式，即可运行分析。

## 支持语言

- English
- 简体中文
- 繁體中文

## 输出模式

- 入门：用更短、更直白的方式说明背景、重点和限制。
- 分析师：保留较完整的证据、机制、多种解释和来源追踪。
- 高管：面向会议和决策讨论的简明情报简报。

## 使用步骤

1. 在浏览器中打开 `dashboard/index.html`。
2. 选择语言。
3. 粘贴文本或上传 `.md` / `.txt` 文件。
4. 选择一个引导问题。
5. 选择入门、分析师或高管模式。
6. 运行分析并查看各个结果部分。

## 重要限制

系统不提供预测、概率、交易建议、投资建议，也不声称确定性。它用于结构化分析、证据查看和后续监测问题整理。
