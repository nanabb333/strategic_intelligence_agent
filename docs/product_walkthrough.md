# Product Walkthrough

## 1. Load A Document

The analyst can paste text into the workbench or upload a Markdown or text file.

## 2. Run Analysis

The dashboard displays collapsible sections:

- Document Summary.
- Scenario Classification.
- Historical Analogues.
- Current Context.
- Implications.
- Executive Brief.

## 3. Review Evidence

Every output area shows source origin labels:

- Input Document.
- Historical Database.
- Context Knowledge Base.

## 4. Export The Brief

The analyst can export Markdown or TXT from the dashboard. The repository
validation flow also generates brief files into `outputs/` and `demo_outputs/`.
