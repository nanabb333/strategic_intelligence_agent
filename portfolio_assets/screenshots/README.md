# Screenshots

Store curated screenshots for portfolio and GitHub presentation.

Preferred screenshot content:

- active project
- decision question
- accepted evidence
- analysis result
- reviewer workspace panels

Avoid screenshots that show only an empty workspace unless documenting first-run behavior.
