# 非 AI 使用者指南

這個工作台面向不會寫提示詞的使用者。選擇語言，貼上或上傳文件，選擇一個引導問題，再選擇輸出模式，即可執行分析。

## 支援語言

- English
- 简体中文
- 繁體中文

## 輸出模式

- 入門：用更短、更直白的方式說明背景、重點和限制。
- 分析師：保留較完整的證據、機制、多種解釋和來源追蹤。
- 高階主管：面向會議和決策討論的簡明情報簡報。

## 使用步驟

1. 在瀏覽器中開啟 `dashboard/index.html`。
2. 選擇語言。
3. 貼上文字或上傳 `.md` / `.txt` 檔案。
4. 選擇一個引導問題。
5. 選擇入門、分析師或高階主管模式。
6. 執行分析並查看各個結果部分。

## 重要限制

系統不提供預測、機率、交易建議、投資建議，也不聲稱確定性。它用於結構化分析、證據查看和後續監測問題整理。
