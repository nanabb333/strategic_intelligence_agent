# Strategic Intelligence Agent Project Brief

## Purpose

Strategic Intelligence Agent is an analyst productivity and executive
decision-support tool. It helps transform unstructured documents into
structured strategic intelligence briefs.

The project evolves the earlier `financial_rubric_agent` concept from:

```text
Document Input -> Rubric Scoring -> AI Writing
```

into:

```text
Document -> Issue Extraction -> Scenario Classification
-> Historical Analogue Retrieval -> Current Context Retrieval
-> Implication Analysis -> Executive Intelligence Brief
```

## Portfolio Position

This repository is designed to demonstrate:

- LLM application development.
- Agent-oriented workflow design.
- Tool use and retrieval integration.
- Multi-step workflow orchestration.
- Business analytics and strategic reasoning.
- Executive-facing synthesis.

## Non-Goals

This is not:

- A trading system.
- A forecasting system.
- An investment recommendation system.

It does not generate buy, sell, hold, timing, or portfolio allocation advice.

## Target User

The primary user is an analyst, strategist, founder, operator, or executive who
needs to convert source material into a concise decision-support brief.

## Primary Output

The core output is an executive intelligence brief that includes:

- Key issues.
- Scenario classification.
- Relevant historical analogues.
- Current context.
- Strategic implications.
- Open questions and uncertainties.
