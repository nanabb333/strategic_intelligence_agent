## Website Guidance

GitHub's Website field should remain empty unless a publicly accessible URL exists. The following address is the local launch URL only and must not be presented as a hosted website or public demo:

```text
http://localhost:8000
```

Compatibility route:

```text
http://localhost:8000/workspace
```
