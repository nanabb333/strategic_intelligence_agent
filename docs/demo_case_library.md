## How To Use This Library

1. Read a demo input in `demo_cases/`.
2. Open the matching folder in `demo_case_outputs/`.
3. Review `analysis.json` for structured fields and `brief.md` for the executive-facing artifact.
4. Use the walkthrough doc to understand what the case demonstrates in the portfolio.

## Evidence Note

The outputs are generated by the local deterministic pipeline. Historical outcomes are simplified educational summaries from local knowledge bases. Evidence credibility notes describe internal confidence and source-status labels, not real-world accuracy.
