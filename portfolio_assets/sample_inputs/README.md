# Sample Inputs

Store curated input examples for product walkthroughs.

Use neutral, non-sensitive examples. Do not include private documents, credentials, personal data, or proprietary source material.

## Recommended Demo Input

Use the V4 workspace demo input:

```text
demo_case_outputs/v4_workspace/question_1_q_export_controls_immediate/input.txt
```

Reviewer question:

```text
Should we adjust investment and supply-chain plans in response to new export controls?
```

Why this works:

- It is strategic rather than conversational.
- It involves evidence, assumptions, and constraints.
- It shows why reviewer-controlled interpretation matters.
- It avoids legal or investment advice framing.
