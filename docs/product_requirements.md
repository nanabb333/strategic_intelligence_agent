# Product Requirements

## Goal

Create a local Analyst Workbench for Strategic Intelligence Agent V2.

## Users

- Strategy analysts.
- Business analysts.
- Risk analysts.
- Product and operations leaders.

## Core Requirements

- Paste document text.
- Upload Markdown or text files.
- Display analysis sections in collapsible panels.
- Show evidence source origins.
- Export executive briefs as Markdown and TXT.
- Preserve non-advisory positioning.

## Non-Goals

- Forecasting.
- Trading recommendations.
- Investment advice.
- Paid API integration.

## Acceptance Criteria

- Dashboard assets exist and load as static files.
- Demo inputs generate outputs.
- Exports can be produced by validation.
- Evidence trace text exists in outputs.
- Forbidden advice language is absent.
