# Portfolio Assets

This folder is reserved for curated product presentation assets.

Use it for materials that help reviewers understand the product quickly:

- screenshots
- sample inputs
- sample outputs
- future demo media

Generated run artifacts should remain under `outputs/` or `demo_case_outputs/`. This folder is for curated portfolio-facing assets only.

## Structure

```text
portfolio_assets/
  screenshots/
  sample_inputs/
  sample_outputs/
  media/
```

## Current Status

The current README hero image uses `docs/Images/dashboard-overview.png`. Future portfolio screenshots should show a populated Decision Workspace with an active project, decision question, evidence, analysis result, and reviewer state.
