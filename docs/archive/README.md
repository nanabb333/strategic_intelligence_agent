# Documentation Archive

This directory preserves superseded milestone, architecture, planning, audit, portfolio, and reviewer documents for repository history.

These files are not the current product contract. Start with the [Documentation Index](../DocumentationIndex.md), [Product Overview](../ProductOverview.md), and canonical [Architecture](../architecture.md).

Archived documents retain the terminology and claims of their original milestones. Their relative links are maintained where practical, but current behavior and version status must be interpreted from the canonical documents above.
