# Resume Bullets

- Built a deterministic Strategic Intelligence Agent that converts source
  documents into executive briefs using issue extraction, scenario
  classification, historical analogue retrieval, current context retrieval, and
  evidence traceability.
- Designed a local Analyst Workbench with paste/upload input, collapsible
  analysis sections, evidence source labels, and Markdown/TXT export.
- Created curated historical and current-context knowledge bases to demonstrate
  retrieval-augmented decision support without paid APIs or forecasting claims.
- Added validation scripts that verify retrieval, generated outputs, evidence
  traces, dashboard assets, and non-advisory language boundaries.
- Packaged the project as a GitHub portfolio case study for agent systems,
  business analytics, workflow orchestration, and executive intelligence.
