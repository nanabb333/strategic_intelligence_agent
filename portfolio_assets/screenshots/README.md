# Screenshots

This directory is the canonical location for portfolio screenshot assets and their status.

Preferred screenshot content:

- active project
- decision question
- accepted evidence
- analysis result
- reviewer workspace panels

Avoid screenshots that show only an empty workspace unless documenting first-run behavior.
