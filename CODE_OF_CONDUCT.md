# Code of Conduct

This project is maintained as a professional portfolio and learning repository.

Participants are expected to:

- Communicate respectfully.
- Keep feedback specific and actionable.
- Avoid personal attacks, harassment, or discriminatory language.
- Respect that the project is a local decision-support tool, not a source of financial, legal, or geopolitical advice.

Maintainers may remove comments or contributions that are abusive, irrelevant, unsafe, or inconsistent with the purpose of the project.
